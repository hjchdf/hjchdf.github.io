概述

一般流程为：面试/笔试(数轮) -> offer -> 三方 -> 劳动合同

注意：很多公司会要求学生先进行全职实习，待毕业后签订劳动合同，一般实习期可以抵扣后面的试用期

关于offer

公司的offer是具有法律效力的要约，即向应聘者发出的签订劳动合同的请求。

根据《中华人民共和国合同法》规定，受要约人有理由认为要约是不可撤销的，并已经为履行合同作了准备工作的，要约不得撤消。应聘者为了实现订立劳动合同的目的，已经辞去了原先的职务，即已经为履行劳动合同作了准备工作。这份要约公司已经不能撤消了。如果要撤消，必须承担因此给应聘者带来的损失。

  • offer具有法律效应

  • 公司毁offer，候选人可以要求赔偿

  • 候选人毁offer，公司也可以要求赔偿，但一般不追究

    1. 《劳动法》立法初衷更多是为了维护劳动者的合法权利不受侵害，其一般在雇佣关系中处于弱势地位
    2. 候选人举证公司容易，公司打官司成本可能更高

  • offer与劳动合同起冲突时，优先看劳动合同（签合同的同学们一定要一条一条过，不清楚的要hr当场解释）

  • 正式的offer应当包含下列内容：

    1. 职位
    2. 薪资，包括基本薪资和绩效的比例等
    3. 合同期限，包括试用期期限
    4. 入职时间
    5. 办公地点
    6. 需要准备的资料
    7. 一般还规定了违约相关处理

  • 对于收到纸质offer的应聘者，要注意offer上是否有有权人的签字或公司盖章。如果应聘者得到的只是一张简单打印的纸，可能出现无法得到认可的情况。

  • 如果公司的offer没有明确入职时间，建议先与公司确认入职时间后再辞职。因为没有明确时间的offer在法律上是可撤消的。

关于三方

前言

三方很重要，但不是非签不可。签三方时一定要明确条款，有疑问及时询问HR。

三方定义

  • 三方是毕业生、用人单位、学校这三方。三方协议即《普通高等学校毕业生、毕业研究生就业协议书》（不同地区命名不一样），是毕业生、用人单位、学校这三方，就学生毕业就业一事所达成的协议，用来明确在就业过程中各自的权利与义务。

  • 三方具有唯一性，即是带有编号的，一人一号，学生不能同时与多个企业签订三方

  • 生效期：签订三方协议（三方均签名盖章确认）起到至学生到用人单位报到并签订劳动合同止。

  • 三方协议不等于劳动合同。

    • 三方协议是确定就业及用人意向，而劳动合同是确定劳资关系。

      法律并没有专门针对对三方协议予以明确的规定，因不存在劳资关系，所以三方协议中的违约金更多地适用于《民法通则》与《合同法》。而劳动合同则是适用于《劳动法》与《合同法》。

    • 三方协议具有法律效力，一旦签署，任何一方不得擅自解除，否则违约方应向权利受损方支付协议条款所规定的违约金。从实际情况来看，多为毕业生违约。

    • 三方协议的主要内容包含报道期限、工作期限（三方协议上的工作期限即为第一次签订劳动合同的工作期限）、试用期限、试用期与转正后的劳动报酬（劳动报酬包括工资及相关福利）、户口问题、社会保险与公积金、无法履约的情况下达成的约定等。

作用

  • 对学生
    1. 保障第一份工作，避免“毕业即失业”
    2. 约定薪酬待遇，保障自身权益
    3. 学校会根据三方协议来办理户籍、档案等一系列关系的转移，不用担心出现“黑户”等情况
  • 企业：
    1. 留住人才，使自己在招聘中的花费有价值
  • 学校：
    1. 统计就业率，同时是见证单位，不承担任何责任
    2. 在学生毕业离校前，学校将根据协议书的内容开具毕业生就业报到证和户口迁移证，同时转递学生档案。 如果毕业生未签定就业协议书，学校将把其关系和档案转递回原籍。
    3. 校友追踪

流程

  • 签署三方需要学生、学校、企业签字/加盖公章。
  • 同时，在签约的过程中，需仔细阅读三方协议详细要求，并在与企业约定并协商好相关条件与要求之后，再进行签署，切勿盲目跟着企业的节奏走。

解约

三方节约较麻烦，会有违约金，也会涉及到个人立场、规划、诚信。

  • 学生解约

    • 一般流程：

      1. 与企业协商，办理解约函，加盖企业公章
      2. 向学校就业中心提交解约函并提出书面申请（可能会需要新公司的录用函）
      3. 学校换发新的三方协议

    • 关于违约金：

      • 很多三方协议中有规定违约金的，大部分情况下，违约金约等于offer上约定的一个月工资，但是考虑到学生的情况，通常情况下违约金不会超过5000人民币。
      • 免除违约金的骚操作
        1. 方法一：和HR沟通，看能不能不交违约金，看人品
        2. 方法二：三方不约束劳动关系，只是合同，按时报到、签订劳动合同，然后辞职即可。（注意：这种做法很伤人品，而且有的单位可能会扣档案，对后面影响很大，推荐好聚好散。）
        3. 注意，方法二必须满足以下3点：
          1. 你想要解约的这家公司的入职日期在前（你不能同时签订两份劳动合同）
          2. 能够在去新公司之前辞职（一般来说试用期提前3天，正式员工提前30天）
          3. 你与新公司协商好，凭发你的offer入职（协商好不签三方），并签订劳动合同

    • 备注：

      • 因升学而导致无法及时去企业报道的情况下属学生违约。
      • 加入国企、事业单位必须签三方以保证档案与组织关系顺利转接。

  • 企业解约

    • 一般而言，很少有企业会违约，毕竟花了一定的人力成本进行校园招聘

    • 如果对方毁约，和毁offer一样，要求赔偿

      可能公司合同有说，本公司有权利根据业务需要调整员工的人事安排。

    • 和毁offer不同，三方部分可能还要涉及户口、档案等的转回

      敦促对方要求把户口等资料在一个星期内转回到学校方便再找工作，将自己的损失降低到最小化

  • 不可抗力因素解约

    一般情况下，因不可抗力而导致三方协议无法继续执行时，双方解约或无责任方解约时无需承担责任，具体条例一般会在三方协议中列出，如：

    • 用人单位被撤销或宣告破产。

    • 学生在报到时未取得毕业资格。

    • 学生被判处拘役以上的刑罚或劳动教养。

    • 法律、法规、政策规定的其他情况。

    • 自然灾难、战争等不可抗力。

注意事项

  1. 备注栏

    • 现行的毕业生就业协议属“格式合同”，但“备注”部分一定要注意，备注里是允许三方另行约定各自的权利义务，一些用人单位备注啥也不写的，就要稍加斟酌是否是在逃避某些责任或义务。

    • 包括签约前达成的工资、奖金、补贴、福利、股票、期权和五险一金等，须仔细备注清楚，为的是以后在发生纠纷时维护自身合法权益。

  2. 先入职再开除你

    • 可以免毁三方的违约金

  3. 三方流程：

    • 用人单位填写完毕、盖章后 -> 学校就业指导中心鉴证盖章
    • 否则单位在填写时，工资待遇等与过去承诺的大相径庭，但和学校都已经签字盖章，回天无力。

关于实习/试用

  • 需要分清的概念有实习期/ 试用期/ 见习期，应届生主要接触的有前2种

  • 实习/试用区别

    1. 两者最大的区别在于是否毕业

      毕业以后从理论上就失去了实习的主体资格，自然也不能进行实习。个别单位会以实习的名义，使用已经毕业的学生进行劳动生产而不签订劳动合同，这是不符合劳动法要求的

    2. 试用期的当事人双方存在着劳动关系，用人单位对劳动者承担无过错责任，与劳动者共同履行缴纳社会保险费用的义务，向劳动者支付的工资报酬不得低于当地最低工资标准。

    3. 学生实习所在的单位对于实习学生，不承担无过错责任，不须执行最低工资标准。

  • 试用期法律规定

  • 注意事项

    • 实习期

      1. 提前结束实习期，除实习协议中另有规定，否则无需支付违约金
      2. 实习期不执行最低工资标准，建议实习之前先确定实习薪酬

    • 试用期

      1. 试用期时长有明确规定（见上图），不能过长

      2. 不合法：单独签订试用期合同/试用期不签订合同

        根据《劳动合同法》第十九条的规定：试用期包含在劳动合同期限内。劳动合同仅约定试用期的，试用期不成立，该期限为劳动合同期限。

        只要从员工用工之日起单位就和员工建立了劳动关系，就属于事实劳动合同关系。不签的话，就要两倍工资支付，一直到第十二个月支付完毕

      3. 员工试用期满后，公司无论出于何种原因，都不能延长试用期。

      4. 试用期辞退员工，企业必须要拿出明确的试用期录用条件，证明员工不符合录用条件

        《劳动合同法》第三十九条规定：“在试用期间被证明不符合录用条件的，用人单位可以解除劳动合同并不给予任何经济补偿”。

      5. 试用期需要缴纳社保。

        《中华人民共和国劳动合同法》第三十八条规定，用人单位有下列情形之一的，劳动者可以解除劳动合同：未依法为劳动者缴纳社会保险费的。

        而且，员工要是以这个理由提出辞职，公司是需要支付经济补偿金的。正常情况下，试用期开始公司就要及时缴纳社保，一般是在员工入职30天内就必须缴纳。

档案和户口

此部分参考：这“档”子事不处理好，高薪offer也不管用！ - 阿熊学长的文章 - 知乎

定义

  • 毕业生的学籍档案是指通过参加全国统一考试并被录取的大中专及以上院校学生的档案。
    • 它以文字资料的形式记录了高考成绩、在校学习成绩、家庭状况、在校期间表现和奖惩情况等。
  • 毕业生的人事档案由学籍档案转换而来，是指毕业生毕业后，与用人单位（含人才）签订就业协议后，在其学籍档案中放入该毕业生的就业通知书。然后由学校将档案转交毕业生就业单位的人事部门或委托的人才交流机构。

性质

毕业生档案是用人单位选拔、聘用毕业生的重要依据。

  1. 内容：《高校毕业生登记表》、《学习成绩表》、在校期间的一切奖惩情况；入团、入党志愿书等政治历史情况的审核材料、毕业离校前的体检表等

  2. 档案袋必须密封保存，密封处加盖接收单位公章，拆开就失效

  3. 国家规定，档案是不能自带的，也不能拆开的，必须挂靠在有人事接收权的单位，例如：人才市场、事业单位、国企、公务员等机构，才能享受人事服务

作用

  1. 转正定级，获取干部身份；

  2. 考研；

  3. 考公务员，过政审关；

  4. 评职称；
  5. 考资格证；
  6. 办理准生证；
  7. 出国审查，过政审关；
  8. 养老保险，计算工龄；
  9. 某些城市，办理居住证；
  10. 进国企，事业单位；
  11. 住房补贴发放；
  12. 入团入党。

毕业后档案去向

  1. 暂留学校

    如果毕业时没有及时找到合适的工作（或考研、考公）可在校申请暂缓就业；

    在两年内找到工作的，及时办理取消暂缓手续，迁至工作单位或挂靠在人才市场。（即使档案暂放学校，保留学生身份，两年后你仍需要迁出档案）

  2. 存放人才市场

    未就业的毕业生，档案挂靠在人才市场，享受人事服务，包括计算工龄、评定职称、一年后转正定级拥有干部身份等等。（拥有干部身份、评定职称、计算工龄还有各种人事服务）

  3. 毕业后档案主动打回原籍

​ 毕业生在毕业前没有主动处理档案，未办理《接收函》给学校，那么学校所开的《报到证》是默认把档案派遣回生源地的。（只是存放档案，容易丢失干部身份，以后迁出来非常麻烦和费钱）

  1. 毕业后档案寄送公司/单位
    • 国有企业、事业单位、部分民营企业、私企等有档案接收管理权，可由单位接收、管理档案；
    • 但绝大部分的民营企业、私企、外企这些都是没有档案接收管理权，要自行处理档案。可到各地公共就业和人才服务机构负责提供档案管理等人事代理服务的部门办理档案挂靠。
  2. 自费留学，档案放在留学服务中心
    • 按相关规定，自费出国留学人员的档案管理属于流动人员人事档案管理一类。一般都存放在留学服务中心或人事代理机构，关于海外学历有专门的认证机构。

*谨记：人事档案不允许个人持有，否则会成为“死档”。*

关于档案挂靠

when

此处广州、深圳等地仅为原文举例

总结：想要一个自己可控的管理档案的地方

  1. 还没找到工作，急于在毕业之前给自己的档案找个好去处，以免被学校送回原籍；
  2. 找到工作了，但是公司单位不解决档案和户口问题；
  3. 读书时户口迁来学校，毕业了学校要求迁出户口，而你户口还想留在广州；
  4. 户口在老家生源地，想要落户广州， 以后买房买车，享受各种社保福利；
  5. 单位要你把档案挂集体，但你想跳槽，怕以后想走走不了；
  6. 之前办理了暂缓就业，现在工作了，要解除暂缓，把户口档案挂到人才市场；
  7. 要出国了，户口还想留广州/深圳等地，免得打回原籍；
  8. 暂缓到期了，档案就要回原籍的往届毕业生。

Why

  • 办理挂靠后，你可以放心到市内外一切非国有单位或无人事主管部门的单位工作，而你的合法权益和应有的社会地位同时得到保障。

  • 应届大学生档案挂靠后，可以享受以下服务：

    1. 档案挂靠在人才市场，一年后即可转正定级拥有干部身份
    2. 连续计算工龄（关系到工资级别及以后的退休工资级别）；
    3. 评定职称（本科毕业满一年认定为助理级）；
    4. 接收党籍、党团关系管理、人事档案管理；
    5. 出具证明材料（即出具与个人档案相关的证明材料）；
    6. 代办社会养老、工伤、生育、失业、医疗保险、住房公积金。

Q&A

  1. 是否必须挂靠档案？
    • 否，但毕业生挂靠档案可以算处理档案最省时省力的方法，且一般城市挂靠档案后会有相应福利（有的可以落户，一线城市的户口不必多说），所以建议挂靠。
  2. 挂靠档案后工作迁移怎么办？
    • 档案户口随时可以迁移。
  3. 挂靠档案，户口怎么办？
    • 档案挂靠，户口随意，但不能只放户口。
    • 办理档案户口后，需要开什么档案的证明或是借户口本办理港澳通、护照等，直接带上抵押证件如四六级证书、学位证、毕业证等过去借，马上就可以拿出来，很简单。
  4. 挂靠收费？
    • 档案挂靠跟档案保管是两个概念。
    • 现在政策规定的是取消档案保管费，而非档案挂靠费用，因档案挂靠包含人事代理服务，属于额外外包服务，并没有减免。
    • 原文广州为例，1500，无年限
